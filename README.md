# Micen 的个人网站

[![website design](/public/images/blogs/website_design.png)](https://qujia.netlify.app)

## 技术栈

- Nuxt：网站框架
- Nuxt content：Markdown CMS
- Nuxt image：图像优化
- Nuxt color mode：暗色模式
- Tailwind CSS：原子化 CSS
- unplugin-icons：图标渲染

## 部署

克隆仓库

```
git clone https://github.com/Micenjia/micen.git
```

进入项目目录

```
cd micen
```

安装依赖

```
pnpm install
```

启动开发服务器

```
pnpm run dev
```

构建生产环境

```
pnpm run build
```
